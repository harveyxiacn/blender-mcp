# 贡献者行为准则

## 我们的承诺

作为成员、贡献者和领导者，我们承诺让每个人都能在友好、安全和热情的环境中参与我们的社区。

我们承诺以有助于建设一个开放、友好、多元化、包容和健康的社区的方式行事和互动。

## 我们的标准

有助于营造积极环境的行为包括：

* 使用友好和包容的语言
* 尊重不同的观点和经验
* 优雅地接受建设性批评
* 关注对社区最有利的事情
* 对其他社区成员表示同理心

不可接受的行为包括：

* 发表侮辱性或贬低性评论，以及人身攻击
* 公开或私下的骚扰
* 未经明确许可发布他人的私人信息
* 其他在专业环境中被合理认为不当的行为

## 执行责任

社区领导者负责明确和执行我们可接受行为的标准，并将对他们认为不当或有害的任何行为采取适当和公平的纠正措施。

## 范围

本行为准则适用于所有社区空间，也适用于个人在公共空间中正式代表社区的情况。

## 执行

不可接受行为的举报可通过 [GitHub Issues](https://github.com/harveyxiacn/blender-mcp/issues) 或直接联系维护者提交。

所有投诉将被及时、公正地审查和调查。

## 归属

本行为准则改编自[贡献者公约](https://www.contributor-covenant.org)，版本 2.1。
