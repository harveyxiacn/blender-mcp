# 02_界面与导航

本文档将帮助你熟悉 Blender 的界面布局和导航操作，学会如何通过 AI 控制视图和编辑器。

## 概述

- **学习目标**: 掌握 Blender 界面结构，熟练使用 3D 视图导航
- **学习成果**: 能够通过 AI 调整视图、切换编辑器、配置偏好设置
- **前置知识**: [01_Blender基础与MCP入门](./01_Blender基础与MCP入门.md)
- **预计学习时间**: 1-2 小时

## 1. Blender 界面概述

### 1.1 默认布局

Blender 启动后，你会看到以下主要区域：

- **顶栏 (Top Bar)**: 包含文件菜单、编辑菜单、渲染菜单等
- **3D 视图 (3D Viewport)**: 中央最大的区域，用于查看和编辑 3D 场景
- **大纲视图 (Outliner)**: 右上角，显示场景中所有对象的层级结构
- **属性编辑器 (Properties)**: 右侧，显示选中对象的详细属性
- **时间线 (Timeline)**: 底部，用于动画制作

### 1.2 工作区 (Workspaces)

Blender 提供了多个预设工作区，针对不同任务优化布局：

- **Layout**: 通用布局，适合建模和场景搭建
- **Modeling**: 建模专用，提供更多建模工具
- **Sculpting**: 雕刻模式
- **UV Editing**: UV 展开和纹理编辑
- **Shading**: 材质和着色器编辑
- **Animation**: 动画制作
- **Rendering**: 渲染设置和预览

### 1.3 编辑器类型

Blender 的界面由多个编辑器组成，每个编辑器都有特定功能：

| 编辑器 | 用途 | 快捷键 |
|--------|------|--------|
| 3D Viewport | 查看和编辑 3D 场景 | - |
| Outliner | 管理场景层级 | - |
| Properties | 编辑对象属性 | - |
| Shader Editor | 编辑材质节点 | - |
| Timeline | 动画时间轴 | - |
| Dope Sheet | 关键帧编辑 | - |
| Graph Editor | 动画曲线编辑 | - |

## 2. 3D 视图导航

### 2.1 视图操作基础

在 3D 视图中，你需要掌握以下基本操作：

**旋转视图**:
- 鼠标中键拖拽
- 小键盘数字键切换视角（1=前视图, 3=侧视图, 7=顶视图）

**平移视图**:
- Shift + 鼠标中键拖拽

**缩放视图**:
- 鼠标滚轮
- Ctrl + 鼠标中键拖拽

### 2.2 视角类型

**透视 (Perspective) vs 正交 (Orthographic)**:
- **透视**: 模拟人眼视角，有近大远小效果
- **正交**: 平行投影，无透视变形，适合精确建模
- 切换: 小键盘 5

**预设视角**:
- 前视图: 小键盘 1
- 右视图: 小键盘 3
- 顶视图: 小键盘 7
- 相机视图: 小键盘 0

### 2.3 视图对齐

**对齐到选中对象**:
- 小键盘句号 (.) - 聚焦到选中对象

**对齐到全部对象**:
- Home 键 - 查看所有对象

## 3. AI 驱动的界面操作

### 3.1 通过 MCP 获取当前状态

虽然 AI 无法直接"看到"你的 Blender 界面，但可以通过工具获取状态信息：

**示例指令**:
> "告诉我当前场景中有哪些对象？"

**AI 内部操作**:
```json
{
  "tool": "blender_object_list",
  "arguments": {}
}
```

**返回信息**:
- 对象名称列表
- 对象类型（MESH, CAMERA, LIGHT 等）
- 对象位置信息

### 3.2 切换工作区

**示例指令**:
> "切换到 Shading 工作区"

**AI 内部操作**:
```python
# 通过 execute_python 实现
import bpy
bpy.context.window.workspace = bpy.data.workspaces['Shading']
```

### 3.3 调整视图设置

**示例指令**:
> "将 3D 视图切换到线框模式"

**AI 内部操作**:
```python
import bpy
for area in bpy.context.screen.areas:
    if area.type == 'VIEW_3D':
        for space in area.spaces:
            if space.type == 'VIEW_3D':
                space.shading.type = 'WIREFRAME'
```

## 4. 编辑器详解

### 4.1 3D 视图编辑器

**主要功能**:
- 查看和编辑 3D 场景
- 选择和变换对象
- 实时预览材质和灯光

**视图着色模式**:
- **线框 (Wireframe)**: 只显示边线
- **实体 (Solid)**: 显示基础着色
- **材质预览 (Material Preview)**: 预览材质效果
- **渲染 (Rendered)**: 实时渲染预览

**工具栏**:
- 左侧: 选择、移动、旋转、缩放等工具
- 右侧: 对象属性和场景设置

### 4.2 大纲视图 (Outliner)

**功能**:
- 显示场景层级结构
- 快速选择对象
- 管理集合和可见性

**常用操作**:
- 点击眼睛图标: 切换可见性
- 点击相机图标: 切换渲染可见性
- 点击箭头: 展开/折叠层级

### 4.3 属性编辑器 (Properties)

**属性面板**:
- **场景属性**: 渲染设置、单位、帧范围
- **渲染属性**: 渲染引擎、采样、输出
- **世界属性**: 背景、环境光
- **对象属性**: 变换、关系、可见性
- **修改器属性**: 修改器堆栈
- **材质属性**: 材质节点和设置

### 4.4 时间线编辑器

**功能**:
- 控制动画播放
- 设置帧范围
- 插入关键帧

**关键元素**:
- 播放/暂停按钮
- 当前帧指示器
- 帧范围设置

### 5.1 常用设置项

**界面设置**:

- 主题颜色

**视图设置**:

- 自动保存间隔
- 默认对象设置

### 5.2 中文界面配置

**通过 AI 配置**:
> "将 Blender 界面语言设置为中文"

**手动配置**:
1. Edit → Preferences → Interface
2. Translation 勾选
3. Language 选择 "Simplified Chinese (简体中文)"
4. 勾选 Interface, Tooltips, New Data

### 5.3 输入设置

**鼠标设置**:
- 选择方式: 左键选择 (推荐) vs 右键选择
- 中键模拟: 适用于没有中键的鼠标

**键盘设置**:
- 快捷键映射: Blender (默认) vs Industry Compatible

### 6.2 完整代码示例

**示例 1: 获取界面信息**

向 AI 发送指令:
```
请告诉我当前 Blender 的版本和渲染引擎
```

AI 会调用:
```json
{
  "tool": "blender_get_info",
  "arguments": {}
```

**示例 2: 切换着色模式**

向 AI 发送指令:

## 8. 最佳实践

### 8.1 常见模式

**模式 1: 快速视角切换**
- 使用小键盘数字键快速切换标准视角
- 适用场景: 需要从多个角度检查模型

**模式 2: 聚焦工作流**
- 选中对象后按小键盘句号聚焦
- 适用场景: 编辑复杂场景中的单个对象

### 8.2 性能考虑

**优化建议**:
- 在复杂场景中使用实体着色模式而非渲染模式
- 隐藏不需要的对象以提高视图性能
- 使用局部视图 (/) 隔离编辑对象

### 8.3 常见陷阱

**陷阱 1: 找不到对象**
- 原因: 对象在视图外或被隐藏
- 解决: 按 Home 键查看全部，或在大纲视图中检查可见性

**陷阱 2: 无法选择对象**
- 原因: 对象被锁定或在不可选择的集合中
- 解决: 检查大纲视图中的锁定图标

```
1. blender_object_list
2. blender_execute_python (设置 shading.type = 'WIREFRAME')

```

### 练习 2: 工作区切换
- **难度**: ⭐
- **目标**: 理解工作区概念
- **任务**: 向 AI 说: "切换到 Shading 工作区，然后再切回 Layout"

**参考答案**:
```python
# 切换到 Shading
bpy.context.window.workspace = bpy.data.workspaces['Shading']

# 切回 Layout
bpy.context.window.workspace = bpy.data.workspaces['Layout']
```

### 练习 3: 偏好设置
- **难度**: ⭐⭐
- **目标**: 学会配置 Blender
- **任务**: 向 AI 说: "将撤销步数设置为 64"

**参考答案**:
```python
import bpy
bpy.context.preferences.edit.undo_steps = 64
```
