## 教程一：创建简单场景

### 目标

创建一个包含桌子、椅子和灯光的简单室内场景。

### 步骤

#### 1. 清理默认场景

向 AI 发送：
```
请清理 Blender 场景，删除默认的立方体，保留相机和灯光
```

#### 2. 创建地板

```
创建一个 10x10 米的地板平面，添加木纹材质
```

#### 3. 创建桌子

```
创建一个简单的桌子：
- 桌面：2m x 1m，厚度 5cm，高度 75cm
- 4 条腿：每条 5cm x 5cm，从桌面四角向下
- 材质：深棕色木质
```

#### 4. 创建椅子

```
在桌子旁边创建一把椅子：
- 座位：45cm x 45cm，高度 45cm
- 靠背：45cm 宽，40cm 高
- 4 条腿
- 材质：与桌子相配的木质
```

#### 5. 调整灯光

```
设置一个温暖的室内灯光环境：
- 将现有灯光改为面光源
- 能量设置为 200W
- 色温设为暖黄色 (0.99, 0.95, 0.85)
- 位置在桌子上方 2.5m 处
```

#### 6. 调整相机并渲染

```
将相机移动到能看到整个场景的位置，朝向桌子
然后渲染一张预览图
```

### 完成效果

你将得到一个包含桌椅的简单室内场景渲染图。

## 教程二：制作 3D Logo

### 目标

创建一个 3D 文字 Logo 并添加材质效果。

### 步骤

#### 1. 创建文字

```
在 Blender 中创建 3D 文字 "BLENDER"，字体大小 2m，居中放置
```

#### 2. 转换为网格并添加厚度

```
将文字对象转换为网格
然后添加实体化修改器，厚度设为 0.3m
添加倒角修改器，宽度 0.02m，分段 3
```

#### 3. 添加材质

```
为 Logo 创建渐变金属材质：
- 基础色：金色 (1.0, 0.8, 0.2)
- 金属度：1.0
- 粗糙度：0.2
```

#### 4. 设置灯光

```
创建三点布光：
1. 主光：在 Logo 左上方 45 度，白色面光源
2. 补光：在右侧，强度较低
3. 轮廓光：在后方，创造边缘高光
```

#### 5. 添加背景

```
创建一个深灰色渐变背景平面
或设置世界环境为纯黑色
```

#### 6. 渲染输出

```
使用 Cycles 引擎，分辨率 1920x1080
采样数 128，启用降噪
渲染 Logo 图像
```

## 教程三：创建简单角色

### 目标

创建一个简单的卡通风格角色（小机器人）。

### 步骤

#### 1. 创建身体

```
创建一个圆角立方体作为机器人身体：
- 尺寸：1m x 0.8m x 1.2m
- 添加倒角修改器使边缘圆滑
- 位置：(0, 0, 1)
```

#### 2. 创建头部

```
创建一个球体作为头部：
- 半径：0.5m
- 位置：(0, 0, 2)
- 添加平滑着色
```

#### 3. 创建眼睛

```
创建两个小球体作为眼睛：
- 半径：0.15m
- 位置：头部前方两侧
- 材质：发光的蓝色
```

#### 4. 创建手臂

```
创建两条简单的手臂：
- 使用圆柱体
- 长度：0.6m，半径：0.1m
- 分别放在身体两侧
```

#### 5. 创建腿部

```
创建两条腿：
- 使用圆柱体
- 长度：0.8m，半径：0.12m
- 放在身体下方
```

#### 6. 添加材质

```
为机器人各部分添加材质：
- 身体和头部：银色金属 (0.8, 0.8, 0.85)
- 手臂和腿：深灰色
- 眼睛：发光蓝色 (0.2, 0.6, 1.0)
```

#### 7. 合并为一体

```
将所有部件设为身体的子对象
方便后续移动和动画
```

## 教程四：制作简单动画

### 目标

为物体创建一个弹跳动画。

### 步骤

#### 1. 创建弹跳球

```
创建一个球体，位置在 (0, 0, 5)
半径 0.5m，命名为 "BouncingBall"
```

#### 2. 设置动画时间线

```
设置动画范围：起始帧 1，结束帧 120
帧率设为 24fps
```

#### 3. 创建下落关键帧

```
在第 1 帧，球体位置 (0, 0, 5)，插入位置关键帧
在第 20 帧，球体位置 (0, 0, 0.5)，插入位置关键帧
```

#### 4. 创建挤压效果

```
在第 20 帧，设置球体缩放为 (1.3, 1.3, 0.6)，插入缩放关键帧
在第 25 帧，设置球体缩放为 (0.8, 0.8, 1.4)，插入缩放关键帧
```

#### 5. 创建反弹关键帧

```
在第 25 帧，球体位置 (0, 0, 0.5)
在第 45 帧，球体位置 (0, 0, 3.5)
在第 65 帧，球体位置 (0, 0, 0.5)
继续添加衰减的弹跳...
```

#### 6. 设置插值曲线

```
将关键帧的插值类型设为 BEZIER
调整曲线使动画更自然
```

#### 7. 预览和渲染

```
播放动画预览
满意后渲染动画序列
```

## 教程五：室内场景布置

### 目标

创建一个完整的现代客厅场景。

### 步骤

#### 1. 创建房间结构

```
创建房间基础结构：
- 地板：8m x 6m
- 三面墙壁：高度 3m
- 一侧留空作为相机视角
```

#### 2. 添加窗户

```
在一面墙上创建窗户：
- 尺寸：2m x 1.5m
- 窗框材质：白色
- 添加玻璃材质
```

#### 3. 创建沙发

```
创建一个 L 型沙发：
- 主体：2.5m x 1m x 0.8m
- 靠背：高 0.4m
- 扶手：宽 0.2m
- 材质：灰色布艺
```

#### 4. 添加茶几

```
在沙发前创建茶几：
- 尺寸：1.2m x 0.6m x 0.4m
- 材质：木质桌面，金属腿
```

#### 5. 添加电视柜和电视

```
创建电视柜和电视：
- 电视柜：2m x 0.4m x 0.5m
- 电视：1.2m x 0.05m x 0.7m（薄板）
- 电视屏幕：黑色发光材质
```

#### 6. 添加装饰物

```
添加装饰品：
- 茶几上放一个花瓶
- 墙上挂一幅画
- 角落放一个落地灯
```

#### 7. 设置灯光

```
设置室内灯光：
1. 主灯：天花板面光源
2. 落地灯：点光源，暖色
3. 窗户自然光：HDRI 或太阳光
```

#### 8. 设置材质

```
为各个对象添加材质：
- 地板：木地板纹理
- 墙壁：浅灰色乳胶漆
- 沙发：布艺纹理
- 茶几：木纹 + 金属
```

#### 9. 调整相机和渲染

```
设置相机：
- 焦距：35mm（广角）
- 位置：能看到整个客厅
- 开启景深，焦点在沙发

使用 Cycles 渲染，分辨率 1920x1080
```

## 提示和技巧

### 1. 分步执行

复杂任务分成多个小步骤执行，便于 AI 理解和执行。

### 2. 具体描述

提供具体的数值和参数，而不是模糊的描述。

**不好的描述**：
```
创建一个大的立方体
```

**好的描述**：
```
创建一个立方体，尺寸 2m x 2m x 2m，位置在原点
```

### 3. 检查结果

每步执行后检查 Blender 中的结果，确认符合预期后再继续。

### 4. 使用相对引用

引用已创建的对象时，使用名称而不是重新描述。

```
将红色材质应用到刚才创建的球体 "MySphere"
```

### 5. 保存进度

重要步骤后保存文件，防止意外丢失。

```
保存当前 Blender 文件到 D:/Projects/my_scene.blend
```
