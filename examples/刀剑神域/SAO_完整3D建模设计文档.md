## 1. 项目概述与技术规范

### 1.1 项目背景

《刀剑神域》(Sword Art Online, SAO) 是以虚拟现实MMORPG为背景的作品。玩家通过NerveGear头盔进入浮游城「艾恩葛朗特」(Aincrad)，该城堡由100层组成，玩家必须攻略所有楼层才能通关。游戏中死亡即现实死亡。

### 1.2 美术风格定义

| 参数 | 规格 |
|------|------|
| **主风格** | TOON/STYLIZED - 动漫赛璐璐风 |
| **渲染引擎** | EEVEE (实时预览) / Cycles (最终渲染) |
| **着色方式** | Shader to RGB → ColorRamp (Constant, 2-3阶) |
| **轮廓线** | Solidify修改器反转法线 / Freestyle |
| **面部阴影** | SDF面部阴影贴图 (可选) |
| **色彩空间** | sRGB, 饱和度适中偏高 |
| **光照** | 主光+补光+轮廓光三点式, 偏暖色调 |

### 1.3 模型技术规范

| 类别 | 面数范围 | UV | 贴图分辨率 |
|------|----------|-----|-----------|
| **主角色** | 15,000-30,000 tris | 1-2 UV set | 2048×2048 |
| **次要角色** | 8,000-15,000 tris | 1 UV set | 1024×1024 |
| **武器/装备** | 1,000-5,000 tris | 1 UV set | 1024×1024 |
| **怪物/Boss** | 10,000-50,000 tris | 1-2 UV set | 2048×2048 |
| **场景建筑** | 5,000-20,000 tris | 1 UV set | 2048×2048 |
| **小道具** | 200-2,000 tris | 1 UV set | 512×512 |

### 1.4 统一命名规范

```
角色: CHR_[名称]_[变体]     例: CHR_Kirito_BlackSwordsman
武器: WPN_[类型]_[名称]     例: WPN_Sword_Elucidator
装备: EQP_[部位]_[名称]     例: EQP_Chest_KoB_Armor
场景: ENV_[层数]_[区域]     例: ENV_F01_StartTown
怪物: MOB_[类型]_[名称]     例: MOB_Boss_Gleameyes
道具: ITM_[类别]_[名称]     例: ITM_Potion_Healing
UI:   UI_[类别]_[名称]      例: UI_Menu_StatusWindow
```

**头部建模规格**

| **刘海** | 碎刘海覆盖前额, 左右分叉 |

| **建模方法** | 面片法: 多层交错面片, 开启Alpha透明; 或体积法: 低多边形发束+SubSurf |
| **材质** | 各向异性高光 (Anisotropic=0.8), 切线方向沿发丝 |

**服装: 黑色风衣套装 (主形态)**

| 部位 | 详细描述 | 材质参数 |
|------|----------|----------|

| **内衬上衣** | 黑色紧身高领内衣 | Base: #1A1A1A, Roughness: 0.7 |
| **腰带** | 黑色宽腰带, 银色扣具 | 皮革质感, Roughness: 0.4 |
| **胸甲/护胸** | 轻型黑色胸甲, 银色边缘 | 金属质感, Metallic: 0.8 |
| **肩甲** | 小型黑色肩甲, 银边 | Metallic: 0.7, 与风衣连接 |
| **手套** | 黑色半指手套, 银色扣 | 皮革, Roughness: 0.5 |
| **裤子** | 黑色直筒长裤 | Base: #0F0F0F, 布料质感 |
| **靴子** | 黑色中筒靴, 银色扣带装饰, 略有跟 | 皮革+金属扣, Roughness: 0.3 |
| **背部剑鞘** | 斜挎背后, 用于放置武器 | 皮革+金属, 可拆卸 |

**服装变体**

| 变体 | 描述 |
|------|------|
| **初始装备** | 蓝灰色简易皮甲+白色衬衣, 新手短剑 |
| **血盟骑士团制服** | 白红色制服(与亚丝娜同款配色), 实际未正式加入 |
| **Alfheim Online (ALO)** | 黑色精灵装束, 精灵耳, 黑色翅膀 |
| **Gun Gale Online (GGO)** | 黑色军事外套, 光剑, 未来感装备 |

**骨骼与动画要点**

| 要点 | 描述 |
|------|------|
| **骨骼数** | ~60根 (含面部) |
| **风衣物理** | 下摆需要布料物理或骨骼链模拟飘动 |
| **剑术动作** | 需预设: 站立持剑、星爆气流斩、二刀流交叉斩 |
| **表情** | Shape Keys: 默认、微笑、愤怒、痛苦、坚定 |

### 7.5 怪物通用死亡效果

所有SAO怪物死亡时统一效果:

| 阶段 | 描述 | 实现方式 |
|------|------|----------|
| **1. 冻结** | 怪物停止动作, 全身出现蓝色裂纹 | Fresnel + 裂纹贴图动画 |
| **2. 碎裂** | 从裂纹处开始碎片化 | Cell Fracture + 刚体物理 |
| **3. 消散** | 碎片化为蓝色多边形粒子上浮消失 | 粒子系统, 蓝色 #80C0FF |
| **4. 掉落物** | 消散完成后, 掉落物品/金币出现 | 物品模型+发光效果 |

```
├── blender_object_create(type="CUBE")          # 创建基础身体方块
├── blender_modifier_add(type="MIRROR")         # 添加镜像修改器
├── blender_modifier_add(type="SUBSURF", levels=2)  # 细分曲面
└── 手动: 进入编辑模式塑形基础人体

阶段2: 头部与面部
├── blender_object_create(type="UV_SPHERE")     # 头部基础球体
├── 手动: 编辑模式雕刻动漫脸型
├── blender_mesh_edit_advanced(op="inset")      # 挖出眼窝
└── 手动: 制作眼球 (UV_SPHERE + 瞳孔贴图)

阶段3: 发型
├── blender_object_create(type="PLANE")         # 发片基础面片
├── 手动: 复制并排列面片组成发束
├── blender_modifier_add(type="SOLIDIFY")       # 给发片厚度
└── 材质: 各向异性头发着色器

阶段4: 服装 (黑色风衣)
├── 复制身体网格作为服装基础
├── 手动: 编辑形状为风衣造型
├── blender_modifier_add(type="SOLIDIFY")       # 布料厚度
├── blender_modifier_add(type="CLOTH")          # 下摆布料物理 (可选)
└── blender_procedural_material(preset="fabric_silk")  # 布料材质

阶段5: 轮廓线
├── blender_outline_effect(method="SOLIDIFY")   # 描边效果
└── 调整轮廓线宽度和颜色

阶段6: 武器
├── blender_object_create(type="CUBE")          # 剑身基础
├── 手动: 编辑为剑型
├── blender_procedural_material(preset="metal_steel")  # 金属材质
└── 手动: 调整为黑色金属参数

阶段7: 骨骼绑定
├── blender_object_create(type="ARMATURE")      # 创建骨架
├── 手动: 摆放骨骼位置
├── 自动权重绑定
└── 手动: 修正权重绘制

阶段8: 表情系统
├── Shape Keys 制作各表情
└── 驱动器连接面部骨骼
```

### 9.2 场景建模流程 (以始城广场为例)

```
阶段1: 地面
├── blender_object_create(type="PLANE", size=200)  # 大型地面
├── 手动: 细分+雕刻地面起伏
└── blender_procedural_material(preset="stone_cobblestone")  # 石板路材质

阶段2: 建筑模块化
├── 制作墙壁模块 (CUBE → 编辑)
├── 制作窗户模块
├── 制作屋顶模块
├── blender_object_duplicate() 复制排列
└── 手动: 微调每栋建筑差异

阶段3: 传送门
├── blender_object_create(type="CYLINDER")      # 柱子
├── blender_object_duplicate() × 12             # 复制12根
├── 手动: 环形阵列排列
├── 中央传送平台: CYLINDER + 发光材质
└── 粒子系统: 蓝色上浮光点

阶段4: 装饰物

阶段5: 光照

```

### 9.3 怪物建模流程 (以碧蓝恶魔为例)

```

├── 手动: 调整为蓝色恶魔皮肤

阶段5: 武器 (蛇纹剑)
├── 建模巨型弯曲大剑

阶段6: 骨骼与动画
├── 完整骨架 (含翅膀骨骼链)
├── 动画: 待机摆翅、横扫攻击、跳跃、嘶吼
└── 死亡碎裂动画序列
```
